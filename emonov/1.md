# Первая группа вопросов

## 1. Дискретная случайная величина: Способ задания

**Определение:**
Дискретная случайная величина (ДСВ) — это величина, которая может принимать конечное или счётное множество значений, каждое из которых имеет определённую вероятность. В отличие от непрерывных величин, дискретные принимают отдельные, чётко определённые значения, например, результат подбрасывания кубика, количество выигрышей в серии игр и т.п.

#### Способы задания дискретной случайной величины:
1. **Перечисление возможных значений и соответствующих им вероятностей:**
   Основной способ задания дискретной случайной величины заключается в указании всех возможных её значений и вероятностей их появления. Пусть $$X$$ — дискретная случайная величина, принимающая значения $$x_1, x_2, ..., x_n$$. Для каждой $$x_i$$ указывается вероятность $$P(X = x_i)$$, причём должны выполняться следующие условия:
   - $$P(X = x_i) \geq 0$$ для всех $$i$$,
   - $$\sum_{i=1}^{n} P(X = x_i) = 1$$.
   
   **Пример:** Подбрасывание кубика. Случайная величина $$X$$ может принимать значения 1, 2, 3, 4, 5, 6, и вероятность каждого значения равна $$\frac{1}{6} $$.
   
   Таблица распределения:
   
$$\begin{array}{|c|c|c|c|c|c|c|}
\hline
X & 1 & 2 & 3 & 4 & 5 & 6 \\
\hline
P(X=x) & \frac{1}{6} & \frac{1}{6} & \frac{1}{6} & \frac{1}{6} & \frac{1}{6} & \frac{1}{6} \\
\hline
\end{array}$$

2. **Задание через функцию вероятностей:**
   Функция вероятностей $$p(x)$$ задаёт вероятность того, что случайная величина примет значение $$x$$. Если случайная величина $$X$$ принимает значения $$x_1, x_2, ..., x_n$$, то её функция вероятностей может быть записана как:
   
   $$p(x) = P(X = x), \text{ где } \sum_{x} p(x) = 1.$$
   
   **Пример:** Для подбрасывания кубика:
   
$$p(x) =
\begin{cases}
\frac{1}{6}, & \text{если } x = 1, 2, 3, 4, 5, 6, \\
0, & \text{иначе}.
\end{cases}$$
   

3. **Функция распределения (функция накопленной вероятности):**
   Функция распределения $$F(x)$$ дискретной случайной величины — это функция, которая находит вероятность того, что случайная величина $$X$$ примет значение, меньшее или равное $$x$$:
   
	$$F(x) = P(X \leq x).$$
   
   Она возрастает ступенчато и принимает значения от 0 до 1. Для любой дискретной случайной величины выполняется:
   - $$F(-\infty) = 0$$,
   - $$F(+\infty) = 1$$.

   **Пример:** Подбрасывание кубика. Для случайной величины $$X$$, принимающей значения от 1 до 6, функция распределения будет следующей:
   
$$F(x) =
\begin{cases}
0, & \text{если } x < 1, \\
\frac{1}{6}, & \text{если } 1 \leq x < 2, \\
\frac{2}{6}, & \text{если } 2 \leq x < 3, \\
\frac{3}{6}, & \text{если } 3 \leq x < 4, \\
\frac{4}{6}, & \text{если } 4 \leq x < 5, \\
\frac{5}{6}, & \text{если } 5 \leq x < 6, \\
1, & \text{если } x \geq 6.
\end{cases}$$

#### Свойства дискретной случайной величины:
1. **Конечность/счётность множества значений.**
   Дискретная случайная величина принимает конечное или счётное количество значений, каждое из которых имеет положительную вероятность.
   
2. **Ненепрерывность функции распределения.**
   Функция распределения дискретной случайной величины меняется на каждом возможном значении величины, образуя скачки.

3. **Сумма вероятностей равна единице.**
   Для всех возможных значений дискретной случайной величины $$X$$ выполняется $$\sum_{i} P(X = x_i) = 1$$.

4. **Значения случайной величины заданы явным образом.**
   В отличие от непрерывных величин, где можно принимать любые значения в определённом интервале, дискретная случайная величина имеет чётко определённые возможные исходы.

Таким образом, дискретная случайная величина — это удобная модель для ситуаций, когда результат эксперимента или явления можно описать с помощью определённых и конечных исходов, каждый из которых имеет свою вероятность.

## 2. Функция распределения непрерывной случайной величины: Определение и свойства

**Определение:**
Функция распределения непрерывной случайной величины (НCВ) — это функция, которая для любого значения $$x$$ показывает вероятность того, что случайная величина примет значение, меньшее или равное $$x$$. Обозначается как $$F_X(x)$$ и определяется следующим образом:

$$F_X(x) = P(X \leq x),$$

где $$X$$ — непрерывная случайная величина, а $$x$$ — любое действительное число.

Непрерывная случайная величина может принимать бесконечное множество значений на каком-то интервале, и вероятность того, что она примет одно конкретное значение, равна нулю. Поэтому вероятность того, что $$X$$ попадает в определённый диапазон, вычисляется как разность значений функции распределения на концах интервала:

$$P(a < X \leq b) = F_X(b) - F_X(a).$$

#### Основные свойства функции распределения:

1. **Непрерывность:**
   Функция распределения непрерывной случайной величины является непрерывной, что означает отсутствие скачков или разрывов в её графике. Это связано с тем, что вероятность точного значения случайной величины равна нулю. 

	$$\lim_{x \to c^-} F_X(x) = \lim_{x \to c^+} F_X(x) = F_X(c) \text{ для любого } c \in \mathbb{R}.$$

2. **Неубывание:**
   Функция распределения всегда неубывает, то есть для любых $$x_1 < x_2$$ выполняется:
   
	$$F_X(x_1) \leq F_X(x_2).$$
   
   Это отражает тот факт, что вероятность попасть в больший диапазон значений всегда больше или равна вероятности попасть в меньший диапазон.

3. **Границы функции распределения:**
   Функция распределения стремится к нулю при $$x \to -\infty$$ и к единице при $$x \to +\infty$$:
   
	$$\lim_{x \to -\infty} F_X(x) = 0 \quad \text{и} \quad \lim_{x \to +\infty} F_X(x) = 1.$$
   
   Это означает, что вероятность того, что случайная величина примет значение меньше бесконечности, равна 1, а вероятность того, что она примет значение меньше минус бесконечности — 0.

4. **Вероятность попадания в точку:**
   Для непрерывной случайной величины вероятность того, что она примет конкретное значение $$P(X = x)$$, равна нулю:
   
	$$P(X = x) = 0.$$
   
   Это связано с тем, что у непрерывной величины множество значений бесконечно, и одно конкретное значение имеет «нулевую» вероятность по сравнению с целым интервалом.

5. **Вероятность попадания в интервал:**
   Вероятность того, что непрерывная случайная величина $$X$$ попадёт в некоторый интервал $$(a, b]$$, можно выразить через значения функции распределения на концах этого интервала:
   
	$$P(a < X \leq b) = F_X(b) - F_X(a).$$
   
   Эта разность определяет вероятность нахождения случайной величины в заданном диапазоне.

6. **Плавность графика функции распределения:**
   График функции распределения имеет плавный характер, без резких изменений. В типичных случаях он имеет S-образную форму, медленно нарастая в начале и конце и быстро изменяясь в середине интервала возможных значений случайной величины.

#### Пример:
Рассмотрим непрерывную случайную величину $$X$$, которая распределена равномерно на интервале $$[a, b]$$. Функция распределения для такой случайной величины будет иметь следующий вид:

$$F_X(x) =
\begin{cases}
0, & x < a, \\
\frac{x - a}{b - a}, & a \leq x \leq b, \\
1, & x > b.
\end{cases}$$

График этой функции — это прямая линия, плавно возрастающая от 0 до 1 на интервале $$[a, b]$$.

#### Связь с плотностью распределения:
Если случайная величина имеет плотность распределения $$f_X(x)$$, то функция распределения $$F_X(x)$$ является её интегралом:

$$F_X(x) = \int_{-\infty}^{x} f_X(t) \, dt.$$

Это означает, что функция распределения описывает вероятность того, что случайная величина примет значение, меньшее или равное $$x$$, а плотность распределения показывает, с какой интенсивностью эта вероятность изменяется на каждом отдельном отрезке значений.

## 3. Плотность распределения непрерывной случайной величины: Определение и свойства

**Определение:**
Плотность распределения непрерывной случайной величины — это функция, которая описывает распределение вероятностей случайной величины на интервале её возможных значений. Плотность распределения $$f_X(x)$$ определяет, насколько «сконцентрирована» вероятность вблизи значения $$x$$. Для любой непрерывной случайной величины $$X$$ плотность распределения удовлетворяет следующим условиям:

$$P(a < X \leq b) = \int_a^b f_X(x) \, dx.$$

Здесь $$P(a < X \leq b)$$ — вероятность того, что случайная величина примет значение в интервале $$(a, b]$$, а $$f_X(x)$$ — плотность распределения в точке $$x$$. Плотность не даёт вероятность конкретного значения $$x$$, так как $$P(X = x) = 0$$ для непрерывной случайной величины, но она даёт представление о том, как распределена вероятность на различных участках числовой оси.

#### Свойства плотности распределения:

1. **Неотрицательность:**
   Для любой точки $$x$$ плотность распределения неотрицательна:
   
   $$f_X(x) \geq 0 \quad \forall x \in \mathbb{R}.$$
   
   Это свойство отражает тот факт, что вероятность — величина неотрицательная, поэтому плотность, как мера концентрации вероятности, также не может быть отрицательной.

2. **Нормировка:**
   Интеграл от плотности по всей числовой оси равен 1, так как вероятность того, что случайная величина примет какое-либо значение на всей оси, равна 1:
   
   $$\int_{-\infty}^{+\infty} f_X(x) \, dx = 1.$$
   
   Это условие гарантирует, что полная вероятность распределена по всей области значений случайной величины.

3. **Вероятность на интервале:**
   Для любого интервала $$(a, b]$$ вероятность того, что случайная величина примет значение в этом интервале, равна интегралу плотности распределения на этом отрезке:
   
   $$P(a < X \leq b) = \int_a^b f_X(x) \, dx.$$
   
   Таким образом, плотность распределения описывает вероятность попадания случайной величины в различные диапазоны значений.

4. **Связь с функцией распределения:**
   Плотность распределения является производной функции распределения:
   
   $$f_X(x) = \frac{d}{dx} F_X(x).$$
   
   Функция распределения $$F_X(x)$$ описывает накопленную вероятность, а плотность $$f_X(x)$$ указывает, с какой скоростью она накапливается в каждой точке.

5. **Гладкость и непрерывность:**
   В большинстве случаев плотность распределения — это непрерывная и гладкая функция. Однако в некоторых случаях плотность может иметь разрывы, например, для кусочно-непрерывных функций (как в случае равномерного распределения).

6. **Форма графика:**
   График плотности распределения показывает «интенсивность» вероятности на разных значениях случайной величины. У некоторых распределений, таких как нормальное, плотность симметрична и имеет форму колокола. У других распределений, таких как показательное, плотность экспоненциально убывает на всём интервале.

#### Примеры плотности распределения:

1. **Равномерное распределение на интервале $$[a, b]$$:**
   Для случайной величины $$X$$, равномерно распределённой на интервале $$[a, b]$$, плотность распределения $$f_X(x)$$ имеет постоянное значение:
   
$$f_X(x) =
\begin{cases}
\frac{1}{b - a}, & a \leq x \leq b, \\
0, & x < a \text{ или } x > b.
\end{cases}$$
   
   Это означает, что вероятность равномерно распределена по интервалу, и каждая точка имеет одинаковую плотность.

2. **Нормальное распределение:**
   Плотность нормальной случайной величины с математическим ожиданием $$\mu$$ и дисперсией $$\sigma^2$$ задаётся формулой:
   
   $$f_X(x) = \frac{1}{\sigma \sqrt{2\pi}} e^{-\frac{(x - \mu)^2}{2\sigma^2}}.$$
   
   Это плотность симметрична относительно $$\mu$$ и имеет «колоколообразную» форму. Чем ближе значение к $$\mu$$, тем выше плотность.

3. **Показательное распределение:**
   Плотность показательной случайной величины с параметром $$\lambda$$ задаётся формулой:
   
$$f_X(x) =
\begin{cases}
\lambda e^{-\lambda x}, & x \geq 0, \\
0, & x < 0.
\end{cases}$$
   
   Плотность экспоненциально убывает с увеличением $$x$$, отражая убывание вероятности на больших значениях.

#### Связь с вероятностью:

Плотность распределения определяет вероятность через интегрирование. Например, для нахождения вероятности того, что случайная величина $$X$$ примет значение в интервале $$[a, b]$$, необходимо проинтегрировать плотность на этом интервале:

$$P(a \leq X \leq b) = \int_a^b f_X(x) \, dx.$$

Плотность даёт более детальную картину распределения вероятностей на числовой оси, показывая, в каких областях вероятность больше, а в каких — меньше.

Таким образом, плотность распределения непрерывной случайной величины — это ключевая функция, описывающая, как вероятность распределена на возможных значениях величины, и служит для вычисления вероятностей различных интервалов.

## 4. Нахождение функции распределения через плотность распределения и наоборот

#### 1. Нахождение функции распределения через плотность распределения

Если известна **плотность распределения** $$f_X(x)$$ непрерывной случайной величины $$X$$, то функцию распределения $$F_X(x)$$ можно найти через интеграл от плотности. 

Функция распределения $$F_X(x)$$ определяет вероятность того, что случайная величина примет значение меньшее или равное $$x$$:

$$F_X(x) = P(X \leq x).$$

В случае непрерывной случайной величины эта вероятность выражается через интеграл от плотности $$f_X(t)$$ по интервалу $$(-\infty, x]$$:

$$F_X(x) = \int_{-\infty}^{x} f_X(t) \, dt.$$

Это выражение показывает, что функция распределения накапливает вероятность до точки $$x$$ и является интегралом плотности на всём промежутке от минус бесконечности до $$x$$.

##### Пример:
Рассмотрим случай, когда плотность распределения $$f_X(x)$$ равномерно распределена на интервале $$[a, b]$$:

$$f_X(x) =
\begin{cases}
\frac{1}{b - a}, & a \leq x \leq b, \\
0, & \text{иначе}.
\end{cases}$$

Функция распределения $$F_X(x)$$ для этого распределения находится по формуле:

$$F_X(x) = \int_{-\infty}^{x} f_X(t) \, dt.$$

Для $$x < a$$:

$$F_X(x) = \int_{-\infty}^{x} 0 \, dt = 0.$$

Для $$a \leq x \leq b$$:

$$F_X(x) = \int_{a}^{x} \frac{1}{b - a} \, dt = \frac{x - a}{b - a}.$$

Для $$x > b$$:

$$F_X(x) = \int_{a}^{b} \frac{1}{b - a} \, dt = 1.$$

Таким образом, функция распределения для равномерного распределения имеет вид:

$$F_X(x) =
\begin{cases}
0, & x < a, \\
\frac{x - a}{b - a}, & a \leq x \leq b, \\
1, & x > b.
\end{cases}$$

#### 2. Нахождение плотности распределения через функцию распределения

Если известна **функция распределения** $$F_X(x)$$ непрерывной случайной величины $$X$$, то плотность распределения $$f_X(x)$$ можно найти как её производную:

$$f_X(x) = \frac{d}{dx} F_X(x).$$

Так как $$f_X(x)$$ является непрерывной и неубывающей функцией, производная $$f_X(x)$$ будет представлять собой плотность вероятности в точке $$X$$, которая показывает, с какой скоростью накапливается вероятность в окрестности этого значения.

##### Пример:
Пусть функция распределения $$f_X(x)$$ равномерной случайной величины на интервале $$[a, b]$$ задана следующим образом:

$$F_X(x) =
\begin{cases}
0, & x < a, \\
\frac{x - a}{b - a}, & a \leq x \leq b, \\
1, & x > b.
\end{cases}$$

Найдём плотность распределения $$f_X(x)$$. Для этого вычислим производную:

Для $$x < a $$ и $$ x > b$$, производная от константной функции равна 0:

$$f_X(x) = 0.$$

Для $$a \leq x \leq b$$:

$$f_X(x) = \frac{d}{dx} \left( \frac{x - a}{b - a} \right) = \frac{1}{b - a}.$$

Таким образом, плотность распределения имеет вид:

$$f_X(x) =
\begin{cases}
0, & x < a \text{ или } x > b, \\
\frac{1}{b - a}, & a \leq x \leq b.
\end{cases}$$

#### Свойства функции распределения и плотности:

1. **Связь через производную и интеграл**: Функция распределения $$f_X(x)$$ — это интеграл от плотности $$f_X(x)$$, а плотность $$f_X(x)$$ — это производная функции распределения $$f_X(x)$$.

2. **Гладкость функции распределения**: Если плотность $$f_X(x)$$ является непрерывной функцией, то и функция распределения $$f_X(x)$$ будет гладкой и непрерывной.

3. **Нормировка**: Плотность распределения всегда интегрируется до 1, а функция распределения принимает значения от 0 до 1:
   
   $$F_X(x) \to 1 \quad \text{при} \quad x \to +\infty, \quad F_X(x) \to 0 \quad \text{при} \quad x \to -\infty.$$

Таким образом, функция распределения и плотность тесно связаны между собой через дифференцирование и интегрирование, что позволяет легко находить одно через другое при наличии полной информации о распределении.

## 5. Математическое ожидание дискретной и непрерывной случайной величины

Математическое ожидание — это одно из основных понятий в теории вероятностей, которое описывает среднее значение случайной величины, если бы мы могли наблюдать её реализацию бесконечное количество раз. Для дискретных и непрерывных случайных величин математическое ожидание вычисляется разными способами.

#### 1. Математическое ожидание дискретной случайной величины

Пусть $$X$$ — дискретная случайная величина, которая принимает значения $$x_1, x_2, \dots$$ с вероятностями $$p_1, p_2, \dots$$, где $$p_i = P(X = x_i)$$.

**Определение:**
Математическое ожидание дискретной случайной величины $$X$$ — это сумма произведений её возможных значений на соответствующие вероятности:

$$\mathbb{E}[X] = \sum_{i} x_i \cdot p_i.$$

Это значение интерпретируется как "среднее" значение, которое примет случайная величина $$X$$ при многократном повторении эксперимента.

##### Пример:
Пусть случайная величина $$X$$ принимает значения 1, 2 и 3 с вероятностями $$P(X = 1) = 0.2$$, $$P(X = 2) = 0.5$$ и $$P(X = 3) = 0.3$$. Тогда математическое ожидание:

$$\mathbb{E}[X] = 1 \cdot 0.2 + 2 \cdot 0.5 + 3 \cdot 0.3 = 0.2 + 1.0 + 0.9 = 2.1.$$

#### Свойства математического ожидания дискретной случайной величины:
1. **Линейность**: Для любых дискретных случайных величин $$X$$ и $$Y$$ и любых констант $$a$$ и $$b$$:
   
   $$\mathbb{E}[aX + bY] = a\mathbb{E}[X] + b\mathbb{E}[Y].$$

2. **Математическое ожидание константы**: Если $$c$$ — константа, то:
   
   $$\mathbb{E}[c] = c.$$

3. **Неотрицательность**: Если $$X \geq 0$$, то и математическое ожидание $$\mathbb{E}[X] \geq 0$$.

#### 2. Математическое ожидание непрерывной случайной величины

Пусть $$X$$ — непрерывная случайная величина с плотностью распределения $$f_X(x)$$. 

**Определение:**
Математическое ожидание непрерывной случайной величины $$X$$ — это интеграл от произведения её возможных значений на плотность распределения:

$$\mathbb{E}[X] = \int_{-\infty}^{+\infty} x \cdot f_X(x) \, dx.$$

Этот интеграл аналогичен сумме для дискретных величин, но в случае непрерывного распределения вероятности "распределены" по всем значениям на числовой оси.

##### Пример:
Пусть $$X$$ — случайная величина, равномерно распределённая на отрезке $$[a, b]$$. Её плотность равна:

$$f_X(x) =
\begin{cases}
\frac{1}{b - a}, & a \leq x \leq b, \\
0, & \text{иначе}.
\end{cases}$$

Математическое ожидание будет:

$$\mathbb{E}[X] = \int_{a}^{b} x \cdot \frac{1}{b - a} \, dx.$$

Решая этот интеграл, получаем:

$$\mathbb{E}[X] = \frac{1}{b - a} \cdot \frac{b^2 - a^2}{2} = \frac{a + b}{2}.$$

Таким образом, математическое ожидание равномерно распределённой случайной величины на отрезке $$[a, b]$$ равно среднему арифметическому концов отрезка.

#### Свойства математического ожидания непрерывной случайной величины:
1. **Линейность**: Как и в случае дискретных случайных величин, математическое ожидание для непрерывных величин тоже линейно:
   
   $$\mathbb{E}[aX + bY] = a\mathbb{E}[X] + b\mathbb{E}[Y].$$

2. **Математическое ожидание константы**: Для константы $$c$$:
   
   $$\mathbb{E}[c] = c.$$

3. **Математическое ожидание положительной случайной величины**: Если $$X \geq 0$$ для всех значений $$x$$, то и $$\mathbb{E}[X] \geq 0$$.

#### Интуитивная интерпретация математического ожидания:
- Математическое ожидание — это «центр тяжести» распределения вероятностей. Оно описывает, какое значение случайной величины можно ожидать в среднем при большом количестве независимых наблюдений.
- Для симметричных распределений, таких как нормальное распределение, математическое ожидание совпадает с медианой и модой.
- Для распределений с сильной асимметрией, таких как показательное распределение, математическое ожидание может значительно отличаться от медианы и моды, отражая "хвост" распределения.

Таким образом, математическое ожидание — это фундаментальная характеристика как дискретных, так и непрерывных случайных величин, которая помогает описать их поведение в среднем.

## 6. Дисперсия дискретной и непрерывной случайной величины

**Дисперсия** — это числовая характеристика, описывающая степень разброса значений случайной величины относительно её математического ожидания. Чем больше дисперсия, тем сильнее разброс значений случайной величины.

#### 1. Дисперсия дискретной случайной величины

Пусть $$X$$ — дискретная случайная величина, которая принимает значения $$x_1, x_2, \dots$$ с вероятностями $$p_1, p_2, \dots$$, где $$p_i = P(X = x_i)$$. Математическое ожидание случайной величины обозначается как $$\mathbb{E}[X]$$.

**Определение:**
Дисперсия дискретной случайной величины $$X$$ — это среднее значение квадрата отклонения случайной величины от её математического ожидания:

$$\text{Var}(X) = \mathbb{E}[(X - \mathbb{E}[X])^2].$$

Другими словами, дисперсия показывает, насколько случайная величина отклоняется от своего среднего значения (математического ожидания). Для вычисления дисперсии можно использовать следующую формулу:

$$\text{Var}(X) = \sum_{i} p_i \cdot (x_i - \mathbb{E}[X])^2.$$

Эта формула проще, так как дисперсия выражается через известные вероятности $$p_i$$ и значения $$x_i$$.

##### Пример:
Пусть случайная величина $$X$$ принимает значения $$1$$, $$2$$, $$3$$ с вероятностями $$0.2$$, $$0.5$$, $$0.3$$, а её математическое ожидание $$\mathbb{E}[X] = 2.1$$.

Тогда дисперсия:

$$\text{Var}(X) = 0.2 \cdot (1 - 2.1)^2 + 0.5 \cdot (2 - 2.1)^2 + 0.3 \cdot (3 - 2.1)^2.$$

Рассчитаем:

$$\text{Var}(X) = 0.2 \cdot 1.21 + 0.5 \cdot 0.01 + 0.3 \cdot 0.81 = 0.242 + 0.005 + 0.243 = 0.49.$$

#### 2. Дисперсия непрерывной случайной величины

Пусть $$X$$ — непрерывная случайная величина с плотностью распределения $$f_X(x)$$, и математическое ожидание $$\mathbb{E}[X] $$ известно.

**Определение:**
Дисперсия непрерывной случайной величины $$X$$ определяется аналогично дискретной величине, но с использованием интеграла:

$$\text{Var}(X) = \mathbb{E}[(X - \mathbb{E}[X])^2] = \int_{-\infty}^{+\infty} (x - \mathbb{E}[X])^2 \cdot f_X(x) \, dx.$$

Это выражение показывает, как вычисляется дисперсия через плотность распределения $$f_X(x)$$.

##### Пример:
Пусть случайная величина $$X$$ равномерно распределена на отрезке $$[a, b]$$. Её математическое ожидание:

$$\mathbb{E}[X] = \frac{a + b}{2}.$$

Дисперсия равномерного распределения:

$$\text{Var}(X) = \int_{a}^{b} (x - \frac{a + b}{2})^2 \cdot \frac{1}{b - a} \, dx.$$

Решая этот интеграл, получаем:

$$\text{Var}(X) = \frac{(b - a)^2}{12}.$$

Таким образом, дисперсия для равномерного распределения на отрезке $$[a, b]$$ пропорциональна квадрату длины отрезка.

#### Свойства дисперсии:

1. **Линейность по константе**: Если к случайной величине добавляется константа, то дисперсия не изменяется:
   
   $$\text{Var}(X + c) = \text{Var}(X).$$
   
   Это связано с тем, что константа сдвигает все значения случайной величины, но не влияет на разброс.
   
2. **Масштабирование**: Если случайная величина умножается на константу $$a$$, то дисперсия изменяется следующим образом:
   
   $$\text{Var}(aX) = a^2 \cdot \text{Var}(X).$$
   
   Таким образом, масштабирование случайной величины влияет на дисперсию квадратичным образом.

3. **Связь с математическим ожиданием квадрата**: Дисперсию можно выразить через математическое ожидание квадрата случайной величины:
   
   $$\text{Var}(X) = \mathbb{E}[X^2] - (\mathbb{E}[X])^2.$$
   
   Это полезное соотношение упрощает вычисление дисперсии в некоторых случаях.

#### Интуитивная интерпретация дисперсии:

- Дисперсия показывает, насколько случайная величина "разбросана" относительно своего среднего значения. Чем больше дисперсия, тем более разбросаны значения.
- Если дисперсия равна нулю, это означает, что случайная величина принимает одно и то же значение с вероятностью 1.
- Для симметричных распределений, таких как нормальное, дисперсия тесно связана с шириной распределения: чем больше дисперсия, тем шире распределение.

#### Дисперсия и стандартное отклонение:
Стандартное отклонение — это квадратный корень из дисперсии:

$$\sigma_X = \sqrt{\text{Var}(X)}.$$

Стандартное отклонение часто используется для более наглядной интерпретации разброса значений случайной величины, поскольку оно имеет те же единицы измерения, что и сама величина $$X$$.

Таким образом, дисперсия — это важная характеристика случайных величин, описывающая их вариабельность относительно среднего значения, как для дискретных, так и для непрерывных распределений.

# Вторая группа вопросов

## 1. Равномерное распределение на отрезке $$[A, B]$$: плотность распределения, математическое ожидание и дисперсия

Равномерное распределение описывает случайную величину, которая принимает значения с равной вероятностью на каком-то интервале. В случае **равномерного распределения на отрезке $$[A, B]$$**, случайная величина может принимать любые значения в этом интервале с одинаковой вероятностью.

#### 1. Плотность распределения

**Определение:** Если случайная величина $$X$$ равномерно распределена на отрезке $$[A, B]$$, то её плотность распределения $$f_X(x)$$ равна:

$$f_X(x) = 
\begin{cases} 
\frac{1}{B - A}, & \text{если } A \leq x \leq B, \\
0, & \text{если } x < A \text{ или } x > B.
\end{cases}$$

Эта функция показывает, что вероятность нахождения случайной величины в любом подотрезке отрезка $$[A, B]$$ пропорциональна длине этого подотрезка. Вне интервала $$[A, B]$$ плотность распределения равна нулю, что отражает невозможность получить значения за пределами этого интервала.

##### Свойства плотности:
- Плотность постоянна на всём интервале $$[A, B]$$.
- Интеграл плотности на всём интервале равен единице:
  
  $$\int_{A}^{B} \frac{1}{B - A} \, dx = 1.$$

#### 2. Математическое ожидание

**Математическое ожидание (среднее значение)** случайной величины $$X$$, равномерно распределённой на отрезке $$[A, B]$$, можно вычислить по следующей формуле:

$$\mathbb{E}[X] = \frac{A + B}{2}.$$

Это значение соответствует середине интервала $$[A, B]$$, так как равномерное распределение подразумевает, что вероятность любого значения на отрезке одинаковая, и в среднем случайная величина окажется посередине.

##### Пример:
Если случайная величина $$X$$ равномерно распределена на отрезке $$[2, 10]$$, то математическое ожидание:

$$\mathbb{E}[X] = \frac{2 + 10}{2} = 6.$$

#### 3. Дисперсия

**Дисперсия** равномерно распределённой случайной величины описывает разброс значений относительно её математического ожидания и вычисляется по следующей формуле:

$$\text{Var}(X) = \frac{(B - A)^2}{12}.$$

Дисперсия пропорциональна квадрату длины интервала $$(B - A)$$, что интуитивно понятно: чем больше длина интервала, тем сильнее разброс возможных значений случайной величины.

##### Пример:
Для равномерного распределения на отрезке $$[2, 10]$$, дисперсия будет равна:

$$\text{Var}(X) = \frac{(10 - 2)^2}{12} = \frac{64}{12} \approx 5.33.$$

#### 4. Стандартное отклонение

Стандартное отклонение — это квадратный корень из дисперсии:

$$\sigma_X = \sqrt{\text{Var}(X)} = \sqrt{\frac{(B - A)^2}{12}} = \frac{B - A}{\sqrt{12}}.$$

Стандартное отклонение даёт наглядную оценку разброса значений случайной величины в тех же единицах, что и сама величина.

#### Итог:
- **Плотность распределения** равна $$\frac{1}{B - A}$$ на интервале $$[A, B]$$.
- **Математическое ожидание** для равномерного распределения: $$\mathbb{E}[X] = \frac{A + B}{2}$$.
- **Дисперсия** равна $$\frac{(B - A)^2}{12}$$, и она увеличивается с ростом длины интервала.
- **Стандартное отклонение**: $$\sigma_X = \frac{B - A}{\sqrt{12}}$$.

Равномерное распределение является простым, но важным базовым распределением, которое часто используется в моделировании равновероятных событий.

## 2. Показательное распределение: плотность распределения, математическое ожидание и дисперсия

**Показательное распределение** описывает время между событиями в процессе, который происходит непрерывно и с постоянной средней скоростью. Часто его используют для моделирования времени ожидания до наступления определённого события (например, времени до следующего отказа оборудования или до прихода следующего клиента).

#### 1. Плотность распределения

Пусть $$X$$ — случайная величина с показательным распределением. Параметром этого распределения является $$\lambda > 0$$, который определяет скорость наступления событий.

**Функция плотности распределения** для показательной случайной величины $$X$$ имеет вид:

$$f_X(x) = 
\begin{cases}
\lambda e^{-\lambda x}, & x \geq 0, \\
0, & x < 0.
\end{cases}$$

Где:
- $$\lambda$$ — параметр распределения, который характеризует частоту событий.
- $$e$$ — основание натурального логарифма (приблизительно 2.718).

##### Свойства плотности:
- Плотность распределения убывает экспоненциально: вероятность того, что событие произойдёт спустя долгое время, уменьшается.
- Интеграл плотности на всём допустимом диапазоне значений ($$[0, \infty)$$) равен единице:
  
  $$\int_0^{\infty} \lambda e^{-\lambda x} \, dx = 1.$$

##### Пример:
Если $$X$$ — время до следующего события с частотой $$\lambda = 2 $$, то плотность распределения:

$$f_X(x) = 2 e^{-2x}, \quad x \geq 0.$$

#### 2. Математическое ожидание

**Математическое ожидание** (среднее значение) для показательной случайной величины $$X$$ с параметром $$\lambda$$ рассчитывается по формуле:

$$\mathbb{E}[X] = \frac{1}{\lambda}.$$

Математическое ожидание показывает, что при показательном распределении среднее время до наступления события обратно пропорционально скорости $$\lambda$$. Чем выше $$\lambda$$, тем чаще происходят события и тем меньше среднее время до события.

##### Пример:
Если $$\lambda = 2$$, то среднее время до наступления события:

$$\mathbb{E}[X] = \frac{1}{2} = 0.5.$$

Это означает, что в среднем событие происходит через 0.5 единицы времени.

#### 3. Дисперсия

**Дисперсия** для показательной случайной величины $$X$$ с параметром $$\lambda$$ вычисляется по формуле:

$$\text{Var}(X) = \frac{1}{\lambda^2}.$$

Дисперсия характеризует разброс значений случайной величины относительно её математического ожидания. Для показательного распределения разброс также обратно пропорционален квадрату частоты $$\lambda$$.

##### Пример:
Для $$\lambda = 2$$, дисперсия будет:

$$\text{Var}(X) = \frac{1}{2^2} = 0.25.$$

#### 4. Стандартное отклонение

**Стандартное отклонение** — это квадратный корень из дисперсии:

$$\sigma_X = \sqrt{\text{Var}(X)} = \frac{1}{\lambda}.$$

Оно описывает типичное отклонение случайной величины от её математического ожидания.

##### Пример:
При $$\lambda = 2$$ стандартное отклонение:

$$\sigma_X = \frac{1}{2} = 0.5.$$

#### 5. Свойства показательного распределения

1. **Безусловная память**: Показательное распределение обладает свойством безусловной памяти. Это означает, что вероятность того, что событие произойдёт спустя $$t$$ единиц времени, не зависит от того, сколько времени уже прошло. Формально:
   
   $$P(X > s + t \mid X > s) = P(X > t).$$
   
   Это свойство делает показательное распределение удобным для моделирования процессов, в которых прошлое не влияет на будущее.

2. **Связь с пуассоновским процессом**: Показательное распределение описывает время между двумя событиями в пуассоновском процессе — процессе, в котором события происходят с постоянной средней скоростью $$\lambda$$.

#### Итог:

- **Плотность распределения** для показательного распределения имеет вид $$f_X(x) = \lambda e^{-\lambda x} $$ для $$x \geq 0$$.
- **Математическое ожидание**: $$\mathbb{E}[X] = \frac{1}{\lambda}$$.
- **Дисперсия**: $$\text{Var}(X) = \frac{1}{\lambda^2}$$.
- **Стандартное отклонение**: $$\sigma_X = \frac{1}{\lambda}$$.

Показательное распределение широко используется в статистике и моделировании для описания времени между независимыми событиями, происходящими с постоянной частотой.

## 3. Нормальное распределение: плотность распределения, математическое ожидание и дисперсия

**Нормальное распределение**, также называемое **гауссовым распределением**, является одним из самых известных и широко используемых распределений вероятностей. Оно описывает множество природных и социальных явлений, таких как рост людей, результаты экзаменов, шум в электронных системах и т.д.

#### 1. Плотность распределения

Пусть $$X$$ — случайная величина, имеющая нормальное распределение с параметрами $$\mu$$ (математическое ожидание) и $$\sigma^2$$ (дисперсия). Её **функция плотности распределения** выражается следующим образом:

$$f_X(x) = \frac{1}{\sigma \sqrt{2\pi}} \exp\left(-\frac{(x - \mu)^2}{2\sigma^2}\right),$$

где:
- $$\mu$$ — среднее значение (математическое ожидание),
- $$\sigma^2$$ — дисперсия,
- $$\sigma$$ — стандартное отклонение,
- $$\exp$$ — экспоненциальная функция.

##### Характеристики функции плотности:
- Плотность распределения симметрична относительно среднего $$\mu$$.
- Пик плотности распределения находится в точке $$\mu$$.
- Распределение убывает к нулю по обе стороны от среднего, и чем дальше от $$\mu$$, тем реже встречаются значения $$x$$.
- Более узкое распределение соответствует меньшему стандартному отклонению $$\sigma$$, а более широкое — большему значению $$\sigma$$.

##### Пример:
Для случайной величины $$X$$, распределённой нормально с параметрами $$\mu = 0$$ и $$\sigma = 1$$ (стандартное нормальное распределение), плотность будет:

$$f_X(x) = \frac{1}{\sqrt{2\pi}} \exp\left(-\frac{x^2}{2}\right).$$

#### 2. Математическое ожидание

**Математическое ожидание** для нормального распределения — это его параметр $$\mu$$, который характеризует центр распределения. Это значение, вокруг которого симметрично распределены все возможные значения случайной величины.

$$\mathbb{E}[X] = \mu.$$

##### Пример:
Если $$X$$ распределено нормально с $$\mu = 10$$ и $$\sigma^2 = 4$$, то математическое ожидание будет:

$$\mathbb{E}[X] = 10.$$

Это означает, что среднее значение случайной величины $$X$$ равно 10.

#### 3. Дисперсия

**Дисперсия** для нормальной случайной величины — это параметр $$\sigma^2$$, который описывает разброс значений вокруг среднего $$\mu$$. Чем больше дисперсия, тем шире распределение и больше разброс значений.

$$\text{Var}(X) = \sigma^2.$$

##### Пример:
Для случайной величины $$X$$, распределённой нормально с $$\mu = 10$$ и $$\sigma^2 = 4$$, дисперсия будет равна:

$$\text{Var}(X) = 4.$$

Чем больше дисперсия, тем сильнее возможные значения отклоняются от среднего $$\mu = 10$$.

#### 4. Стандартное отклонение

**Стандартное отклонение** — это квадратный корень из дисперсии:

$$\sigma_X = \sqrt{\text{Var}(X)} = \sigma.$$

Оно характеризует типичное отклонение случайной величины от её среднего значения.

##### Пример:
Если $$\sigma^2 = 4$$, то стандартное отклонение $$\sigma = \sqrt{4} = 2$$.

#### 5. Свойства нормального распределения

1. **Симметрия**: Нормальное распределение симметрично относительно математического ожидания $$\mu$$.
2. **68-95-99.7 правило**: Это эмпирическое правило для нормального распределения:
   - 68% всех наблюдений находятся в пределах одного стандартного отклонения от среднего $$(\mu - \sigma, \mu + \sigma)$$,
   - 95% — в пределах двух стандартных отклонений $$(\mu - 2\sigma, \mu + 2\sigma)$$,
   - 99.7% — в пределах трёх стандартных отклонений $$(\mu - 3\sigma, \mu + 3\sigma)$$.

3. **Сложение нормальных случайных величин**: Если $$X_1 \sim N(\mu_1, \sigma_1^2)$$ и $$X_2 \sim N(\mu_2, \sigma_2^2)$$ — независимые нормально распределённые случайные величины, то их сумма также будет нормально распределённой:
   
   $$X_1 + X_2 \sim N(\mu_1 + \mu_2, \sigma_1^2 + \sigma_2^2).$$

#### Итог:

- **Плотность распределения** нормальной случайной величины $$X$$ с параметрами $$\mu$$ и $$\sigma^2$$: 
  
  $$f_X(x) = \frac{1}{\sigma \sqrt{2\pi}} \exp\left(-\frac{(x - \mu)^2}{2\sigma^2}\right).$$

- **Математическое ожидание**: $$\mathbb{E}[X] = \mu $$.
- **Дисперсия**: $$\text{Var}(X) = \sigma^2 $$.
- **Стандартное отклонение**: $$\sigma_X = \sigma $$.

Нормальное распределение используется в широком спектре приложений благодаря своим математическим свойствам и удобству при обработке данных, поскольку многие случайные процессы можно описать нормальным распределением.
